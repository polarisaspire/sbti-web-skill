## 关于 SBTI

### 它是什么

SBTI 是一个基于 15 维度、26 种人格类型的娱乐性人格测试。题目风格粗粝毒舌，人格描述有文采，结果天然适合截图分享。每种人格都有一个荒诞的中文名——「死者」「愤世者」「吗喽」「小丑」「废物」——每一个都是当代年轻人的自嘲标签。

### 为什么它能火

2026 年 4 月 9 日，B站一条视频首发，**不到 10 小时席卷全平台**。原因很简单：

- **精准戳中自嘲情绪**：人格标签不是 INFJ/ENFP 这类术语，而是你每天在朋友圈发的那些词
- **题目足够抽象有梗**：问你便秘坐马桶 30 分钟的感受，问你看到萌小孩递糖的第一反应
- **描述文案真的有文采**：SCP 报告体、宇宙级比喻、游戏通关叙事，每个人格都写得用心
- **社交货币属性极强**：一张结果卡片就是一个社交话题，"你是什么 SBTI"取代了"你是什么 MBTI"
- **隐藏彩蛋**：「DRUNK / 酒鬼」需要在特定题目选择"将白酒灌在保温杯当白开水喝"才能触发
- **网站崩了反向营销**：原站挤崩、镜像涌现、"你也打不开吗"成了新的社交话题

## 这个 Skill 能做什么

本项目将 SBTI 测试改造为 **AI 对话体验**，支持四种玩法：

| 模式 | 触发方式 | 说明 |
|------|---------|------|
| 对话式测评 | `/sbti` 或"帮我测 SBTI" | 5 轮毒舌对话完成 30 题，后台默默评分，最终输出人格卡片 |
| 反向人格诊断 | `/sbti 分析 <文本>` | 贴聊天记录 / 朋友圈 / 吵架发言，AI 反推 SBTI 人格 |
| 毒舌人格陪聊 | `/sbti 陪聊 SHIT` | 以指定人格的视角和语气跟你聊天 |
| Prompt 生成 | `/sbti prompt DEAD` | 生成可复用的角色扮演 prompt 模板 |

测试覆盖 **30 题 × 15 维度 × 26 种人格**，含隐藏人格 DRUNK（酒鬼）和兜底人格 HHHH（傻乐者）。

## 复刻网站（`assets/index.html`）

离线单文件，打开即用，包含三个页面：

| 页面 | 说明 |
|------|------|
| **首页** | 进入测试 / 进入人格百科 |
| **答题页** | 32 题（含饮酒彩蛋）+ 进度条，全选后提交 |
| **结果页** | 角色图 + 人格类型 + 匹配度 + 15 维度解读 |

点击首页「人格百科」进入人格类型索引：

- 顶部 26 张角色缩略卡网格，点击跳转对应详情
- 下方完整详情卡列表，每张含：人格标题、标签（标准 / 隐藏 / 兜底）、一句话简介、完整描述
- 隐藏人格 DRUNK 和兜底人格 HHHH 均在列，标签有区分
